==============
 APEX GIF MAKER
==============

○ 소개 (Introduction)

APEX GIF MAKER는 Flydigi APEX 시리즈 게임 컨트롤러의 스크린에 사용할 GIF 애니메이션을 제작하기 위해 만들어진 프로그램입니다. 사용자는 가지고 있는 동영상 파일에서 원하는 구간을 손쉽게 선택하여 컨트롤러에 최적화된 고품질 GIF 파일을 만들 수 있습니다.


● 핵심 기능 (Key Features)

- 간편한 GIF 변환: MP4, MOV, MKV 등 다양한 동영상 파일을 GIF 애니메이션으로 변환합니다.
- 직관적인 구간 선택: 썸네일 미리보기를 지원하는 타임라인 슬라이더와 정확한 시간 입력을 통해 원하는 구간을 쉽게 지정할 수 있습니다.
- 실시간 프리뷰: 선택한 구간의 시작과 끝 프레임을 실시간으로 확인하며 작업할 수 있습니다.
- 다양한 생성 옵션:
    - 해상도 및 FPS: 원하는 GIF의 크기와 초당 프레임 수를 자유롭게 설정할 수 있습니다.
    - 스케일 모드: 원본 영상 비율을 유지(레터박스)하거나, 꽉 채우도록(크롭) 설정할 수 있습니다.
    - 디더링: GIF의 256색 한계를 보완하여 색상 품질을 높이는 디더링 옵션을 제공합니다.
- FFmpeg 자동 설치: GIF 생성에 필수적인 FFmpeg가 PC에 없는 경우, 프로그램이 자동으로 다운로드하고 설정합니다.
- 설정 기억: 마지막으로 사용한 출력 폴더 및 생성 옵션을 기억하여 다음 실행 시 자동으로 불러옵니다.


◎ 사용법 (How to Use)

1. '열기...' 버튼을 클릭하여 GIF로 만들고 싶은 동영상 파일을 불러옵니다.
   (프로그램 첫 실행 시 FFmpeg가 없다면 자동으로 다운로드가 진행됩니다.)

2. 영상이 로드되면, 상단 프리뷰와 하단 썸네일 타임라인이 표시됩니다.

3. 하단의 타임라인 슬라이더를 마우스로 드래그하거나, 상단 프리뷰의 시간 입력창에 직접 시간을 입력하여 GIF로 만들 구간을 선택합니다.
   (선택 구간은 최소 1초, 최대 15초입니다.)

4. '옵션' 영역에서 원하는 GIF의 모드, FPS, 가로/세로 크기, 스케일, 디더링 방식을 설정합니다.

5. '출력' 영역의 '저장 위치...' 버튼을 클릭하여 생성될 GIF 파일의 이름과 저장될 경로를 지정합니다.
   (지정하지 않으면 영상 파일 이름으로 자동 생성됩니다.)

6. 'GIF 생성' 버튼을 클릭합니다. 변환이 완료되면 저장된 경로에 GIF 파일이 생성됩니다.

7. '구간 재생' 버튼을 누르면 현재 선택된 구간만 잘라서 동영상 플레이어로 미리 볼 수 있습니다.

8. 하단 로그창의 '로그 지우기' 버튼으로 작업 기록을 지우거나, '정보' 버튼으로 프로그램 정보를 확인할 수 있습니다.
